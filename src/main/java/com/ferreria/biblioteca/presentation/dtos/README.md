# **Paquete: dto**

## Descripción
Define los **Data Transfer Objects (DTO)** utilizados para la transferencia de datos entre las capas de la aplicación.

## Archivos principales
- * `UserDTO.java` *: Representa los datos del usuario que se transfieren al cliente.
- * `BookDTO.java` *: Contiene información de libros que se envía o recibe.
- **`LoanDTO.java`**: Representa la información de préstamos para la API.

## Función
Separar la lógica interna de las entidades y proporcionar datos optimizados al cliente.
